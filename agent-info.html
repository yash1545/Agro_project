Export Agent Information

Connect with authorized export agents for shipment and documentation support.

Role of Export Agent

Export agents assist farmers in handling shipment, customs clearance, and required export documentation. They operate independently and are not directly managed by the platform.

Services Provided

  • International shipment handling
  • Customs clearance support
  • Export documentation guidance
  • Coordination with logistics providers

How Farmers Can Connect

After order confirmation, farmers can contact export agents using the details shared by the platform. The platform only provides agent information and does not participate in shipment execution.
